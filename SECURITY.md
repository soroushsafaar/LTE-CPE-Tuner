# Security and privacy

This project is for authorized testing of your own LTE/TD-LTE CPE/router from an already-authenticated browser session.

Do not publish:

- your precise home coordinates or address
- IMEI, ICCID, IMSI, SIM serial numbers, public IPs, admin session IDs
- full raw dumps that include device serial numbers
- credentials, cookies, or router backup files

The browser tools use same-origin requests from the router admin page. They do not bypass login, brute-force credentials, unlock SIM/network restrictions, or attack third-party systems.

For public GitHub issues, redact identifiers first.
